# 01. Product

## Что строим

Личный трекер здоровья: вес, КБЖУ, состав тела, шаги, сон, настроение, тренировки. Один экран с правдой за день, графики за период, умные подсказки в нейтральном тоне.

## Целевой юзер

Взрослый который сам разбирается в фитнесе/питании. Не нуждается в коучинге. Хочет одно место для всех данных, быстрый ввод, честные подсказки (не мотивашки), полную свободу в задании целей.

## Чем НЕ являемся

- Не MyFitnessPal (нет огромной базы продуктов и штрих-кодов)
- Не MacroFactor (не подгоняем TDEE адаптивно — юзер задаёт цели сам)
- Не Hevy (не справочник упражнений)
- Не медицинское ПО (никаких диагнозов, скрининга РПП)

## Главные ценности

1. **Скорость ввода:** запись приёма или веса = 3 тапа максимум
2. **Один экран с правдой:** главный экран показывает текущий день целиком, без переключения вкладок
3. **Гибкость над предписаниями:** юзер задаёт КБЖУ, приложение определяет режим
4. **Честный тон:** факт + прогноз, без оценок «молодец/плохо»
5. **Тренд важнее идеала:** везде графики, сравнение «было → стало»

## User stories (MVP)

1. **Утренний замер.** Встал на весы → ввёл вес и состав тела → видит тренд за неделю
2. **Запись приёма.** FAB → Приём → выбрал блюдо из своих → ввёл граммы → авторасчёт КБЖУ
3. **Проверка статуса днём.** Открыл → видит "1247 / 1700, прогноз 1680, дефицит будет −1035"
4. **Перепланирование.** Воскресенье → создаёт новую цель на следующую неделю → автоматически применяется
5. **Просмотр прогресса.** Stats → видит графики за месяц, сравнение начало→конец
6. **Переключение аккаунта.** Tap на аватар → выбирает второй аккаунт (без логина, токен в IDB)
7. **Забытый день.** Свайпнул назад → пустой день → записал задним числом
8. **Diet break.** После 8 недель сушки → новая цель на 10 дней с пометкой break → режим меняется на maintenance, через 10 дней автоматически возвращается старая цель

## Не-цели MVP

Push, синхронизация с Apple Health/Google Fit, чтение весов по Bluetooth, фото-распознавание еды, штрих-коды, рецепты, соц-фичи, экспорт/импорт CSV, i18n, streaks/achievements, AI-чат, справочник упражнений с подходами.
